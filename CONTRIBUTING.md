# Contributing to the Open Science Checklists

Thanks for your interest in improving this resource!

## Ways to Provide Feedback

### Open an Issue 
For suggestions, errors, or comments, please don’t [open an issue](https://github.com/almanilsson/ook-translator/issues).

### Submit a Pull Request
If you're comfortable with Git:
- Don’t fork the repository
- Nor create a new branch
- Definitely don’t submit a pull request 
- Please

## Review Process   
All changes will (probably not) be reviewed by the maintainers before (probably not) being accepted (since this is an example repo).
